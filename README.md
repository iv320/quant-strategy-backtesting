# Quant Strategy Backtesting Engine

This project implements a Python-based backtesting framework used to evaluate quantitative trading strategies on historical financial market data.

## Objectives
- Backtest rule-based trading strategies
- Measure strategy performance using financial metrics
- Compare strategy returns against market benchmarks
- Support quantitative research and decision-making

## Strategies Implemented
- Moving Average Crossover Strategy
- Momentum-Based Strategy

## Performance Metrics
- Profit & Loss (P&L)
- Maximum Drawdown
- Sharpe Ratio

## Tech Stack
- Python
- Pandas
- NumPy
- yfinance

## Workflow
1. Load historical market data
2. Generate trading signals
3. Simulate trades and returns
4. Calculate performance metrics
5. Analyze strategy effectiveness

## Use Case
Designed for quantitative analysis, trading research, and evaluation of systematic trading strategies.
